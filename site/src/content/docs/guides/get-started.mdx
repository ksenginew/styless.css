## Install via package manager

Install Styless’s source Sass and JavaScript files via npm. Package managed
installs don’t include documentation or our full build scripts.
<Tabs>
	<TabItem label="npm">
  ```sh
  npm install styless.css
  ```
  </TabItem>
	<TabItem label="pnpm">
  ```sh
  pnpm add styless.css
  ```
  </TabItem>
  <TabItem label="yarn">
  ```sh
  yarn add styless.css
  ```
  </TabItem>
</Tabs>
